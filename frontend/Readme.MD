# Bridesmaid Color Picker Frontend

## Setup

1. Place your `loading.gif` in `public/`.
2. Install dependencies:
   ```
   npm install
   ```
3. Start the app:
   ```
   npm start
   ```
- Runs on `http://localhost:3000` (or `5173` if using Vite).
- Make sure backend is running on `http://localhost:4000`.
